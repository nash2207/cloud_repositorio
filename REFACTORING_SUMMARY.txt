================================================================================
DEPENDENCY INJECTION
================================================================================

The Genetic Algorithm is now COMPLETELY INFRASTRUCTURE-AGNOSTIC:

❌ BEFORE: GA knew about workers, VMs, CPU, RAM, disk
✓ AFTER:  GA only knows about:
          - Integer arrays (chromosomes)
          - Black-box fitness function (callback)
          - Maximization objective

Example usage:
    def my_fitness_func(chromosome):
        # Custom fitness evaluation
        return score
    
    solution, fitness, history = genetic_algorithm(
        chromosome_length=10,
        gene_range=(0, 2),
        fitness_func=my_fitness_func,  # ← Injected callback
        ...
    )
